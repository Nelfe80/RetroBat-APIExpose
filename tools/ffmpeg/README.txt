ffmpeg tool folder / Dossier outil ffmpeg
=========================================

EN
--
Purpose:
ffmpeg is used locally by APIExpose maintenance/media workflows when video,
audio, thumbnails, or diagnostic captures need to be inspected or converted.

Installation:
1. Download or restore a Windows ffmpeg build from your trusted release archive.
2. Copy the ffmpeg files into APIExpose/tools/ffmpeg.
3. The expected executable path is:
   APIExpose/tools/ffmpeg/ffmpeg.exe
4. Do not keep ffmpeg.exe inside a bin subfolder. If the downloaded archive
   contains bin/ffmpeg.exe, copy ffmpeg.exe up to APIExpose/tools/ffmpeg.
5. Keep the binaries local. Git only tracks this README.txt.
