ImageMagick tool folder / Dossier outil ImageMagick
===================================================

EN
--
Purpose:
ImageMagick is used locally by APIExpose media-maintenance workflows for image
inspection, conversion, resizing, and cleanup.

Installation:
1. Download or restore a Windows ImageMagick portable/install archive.
2. Copy the extracted files into APIExpose/tools/imagemagick.
3. The expected executable path is:
   APIExpose/tools/imagemagick/convert.exe
4. Do not keep convert.exe inside a bin subfolder. If the archive contains
   bin/convert.exe, copy convert.exe up to APIExpose/tools/imagemagick.
5. Keep binaries, delegates, and cache files local. Git only tracks this
   README.txt.

FR
--
Role :
ImageMagick est utilise localement par les workflows de maintenance media
d'APIExpose pour inspecter, convertir, redimensionner et nettoyer les images.

Installation :
1. Telecharger ou restaurer une archive Windows portable/install d'ImageMagick.
2. Copier les fichiers extraits dans APIExpose/tools/imagemagick.
3. Le chemin attendu de l'executable est :
   APIExpose/tools/imagemagick/convert.exe
4. Ne pas garder convert.exe dans un sous-dossier bin. Si l'archive contient
   bin/convert.exe, copier convert.exe directement dans
   APIExpose/tools/imagemagick.
5. Garder les binaires, delegates et caches en local. Git ne suit que ce
   README.txt.
