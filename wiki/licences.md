# Licences

**RetroBat reste un logiciel libre.** APIExpose est un plugin indépendant, licencié séparément — il ne fait pas de RetroBat un produit payant.

## Usage personnel

L'utilisation **personnelle, éducative, communautaire et non commerciale** est libre. Une machine personnelle = une licence personnelle APIExpose.

## Usage commercial

Toute utilisation commerciale nécessite une **licence écrite** : bornes d'arcade, mini-PC, disques durs, cartes SD ou clés USB préconfigurés, packs numériques, distributions RetroBat préconfigurées, services d'installation ou d'assistance payants — et plus généralement toute offre payante où APIExpose est inclus, mis en avant, configuré ou utilisé.

```text
1 machine vendue = 1 licence revendeur APIExpose
```

Un revendeur ne peut ni copier une licence personnelle sur plusieurs machines, ni masquer les informations de licence APIExpose.

## Le Data Pack

Les fichiers de données — XML, fichiers `.MEM`, mappings, définitions RAM, outputs, panels, profils et métadonnées — constituent l'**APIExpose Data Pack**, protégé par une licence dédiée (`DATA-LICENSE.md`). Il est distribué avec les releases pour l'usage personnel ; sa redistribution commerciale suit les mêmes règles que le plugin.

## Les textes complets

Dans le dépôt et dans chaque archive de release :

- `LICENSE.md` — la licence générale ;
- `PERSONAL-LICENSE.md` — l'usage personnel ;
- `COMMERCIAL-LICENSE.md` — l'usage commercial et revendeur ;
- `DATA-LICENSE.md` — le Data Pack ;
- `NOTICE.md` — la notice pour les revendeurs.

Les outils tiers embarqués (ffmpeg, ImageMagick, translateLocally) restent sous leurs licences respectives.
