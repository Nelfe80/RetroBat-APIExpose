# Definitions Dynamic Panel

Deposez ici les definitions CPO/control panel.

Defaults systeme :

```text
resources/dynpanels/systems/<system>.json
```

Overrides jeu arcade :

```text
resources/dynpanels/games/<rom>.json
```

APIExpose utilise ces fichiers pour exposer les couleurs des boutons, leurs
fonctions et les types de controllers aux themes, panels LEDs et clients
WebSocket.
