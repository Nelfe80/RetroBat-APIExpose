# Dynamic Panel Definitions

Put CPO/control panel definitions here.

System defaults:

```text
resources/dynpanels/systems/<system>.json
```

Arcade game overrides:

```text
resources/dynpanels/games/<rom>.json
```

APIExpose uses these files to expose button colors, button functions and
controller types to themes, LED panels and WebSocket clients.
